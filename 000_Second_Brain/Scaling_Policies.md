# Scaling Policies

Scaling policies define the scaling actions that the [[Auto Scaling Groups]] should take in response to conditions or alarms. These policies are based on the following metrics:
	CPU Utilisation
	Memory Usage
	Custom Metrics
Types of policies include
	[[Target Scaling]]
	[[Step Scaling]]
	[[Scheduled Scaling]]
